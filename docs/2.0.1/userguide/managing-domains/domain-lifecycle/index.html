Domain life cycle

Learn how to start, stop, restart, and scale the domain’s servers.

    Startup and shutdown

    There are properties on the domain resource that specify which servers should be running and which servers should be restarted. To start, stop, or restart servers, modify these properties on the domain resource.

    Restarting

    This document describes when to restart servers in the Oracle WebLogic Server in Kubernetes environment.

    Scaling

    The operator provides several ways to initiate scaling of WebLogic clusters.
